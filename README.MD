# Base64 File Encoder

## Simple desktop application, created just to practice Tkinter GUI and Python OOP.

- ### It can encode up to 1000 files at once.
- ### The most common image file formats are supported (PDF, JPG, PNG, SVG, BMP, GIF, TIFF, ICO).
